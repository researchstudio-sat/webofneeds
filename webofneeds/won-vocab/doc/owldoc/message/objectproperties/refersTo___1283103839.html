Object Property: refersTo

https://w3id.org/won/message#refersTo

Annotations (1)

  • comment "Points to another message. Messages always exist in two copies: one on the sender's WoN node and one on the receiver's WoN node. The receiver's WoN node copies the message, mints a new URI for it and points to the original message using 'msg:refersTo'. In case of a forwarded message, this property points to the original message." (en)
